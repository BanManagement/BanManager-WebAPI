# Setup
- Run `npm install`
- Run `node bin/bm-cli setup` and follow instructions
- Run `node server.js`

# Development Setup
- Setup BanManager Minecraft plugin with Spigot
- See Setup instructions above
- For running tests, ensure DB_USER has permission to create databases
